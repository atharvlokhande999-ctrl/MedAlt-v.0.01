💊 MedSearch: AI-Based Medicine Recommendation System

🚀 GitHub Competition + Flask Web Application

---

📌 Overview

MedSearch is an AI-driven system designed to help users find affordable alternatives for medicines.

This project is built as a real-world GitHub competition, where participants develop models to recommend cost-effective medicines based on pharmaceutical data.

It combines:

- 🌐 A Flask Web App (for user interaction)
- 🧠 A Machine Learning Competition System
- 🏆 A Leaderboard-based evaluation pipeline

---

🎯 Objectives

- Help users find cheaper medicine alternatives
- Simulate a real-world AI competition environment
- Encourage development of ML-based recommendation systems
- Promote healthcare affordability using AI

---

🧠 Problem Statement

Given a dataset of medicines, participants must:

👉 Predict affordable alternatives or estimate medicine prices accurately.

---


⚙️ Tech Stack

- 🐍 Python
- 🌐 Flask
- 📊 Pandas, NumPy
- 🔐 Cryptography (for secure evaluation)
- ⚙️ GitHub Actions (automation)

---

🌐 Web Application (Demo UI)

The project includes a Flask-based interface where users can:

- Enter a medicine name
- Get recommended alternatives
- View price comparisons

▶️ Run the app:

cd app
python app.py

---

🏆 Competition Workflow

1. 📥 Participants download dataset from "data/"
2. 🧠 Build models using "starter_code/"
3. 📤 Submit predictions in "submissions/"
4. ⚙️ Scoring script evaluates submissions
5. 📊 Leaderboard updates automatically

---

📦 Submission Format

Participants must submit a ".csv" file:

brand_name,price_inr
Crocin,45
Paracetamol,30

---

🔐 Security (Encryption)

- Public key is shared for secure submissions
- Hidden labels are protected
- Ensures fair evaluation

---

📊 Leaderboard

- Automatically updated
- Stored in "docs/leaderboard.csv"
- Displays ranking of participants

---

🚀 Setup Instructions

1. Clone the repository

git clone <https://github.com/Atharva-U12/Medicine_Recommendation_Challenge.git>
cd MedSearch-Competition

2. Create virtual environment

python -m venv venv
source venv/bin/activate

3. Install dependencies

pip install -r requirements.txt

---

## 🧪 Submission Instructions

1. Fork this repository
2. Improve the model in `startercode/baseline_model.py`
3. Add a file named `results.txt` in the root directory

Format:
username: your_github_username
score: your_score

4. Create a Pull Request

🏆 Leaderboard updates automatically after PR merge
